# Birthday App (Next.js)

A small Next.js app that displays a modern birthday wish card.

Getting started:

1. Install dependencies

```bash
npm install
```

2. Run the dev server

```bash
npm run dev
```

Open http://localhost:3000 in your browser.

Notes:
- This scaffold uses Next.js 14 and React 18.
- Replace the wish text in `pages/index.js` to customize the message.
